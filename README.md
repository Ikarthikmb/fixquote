# FixQuote — Repair Estimator Chrome Extension

Build repair estimates in seconds while you browse. Add parts from Amazon, eBay, PhoneLCDParts, MobileSentrix and anywhere else — price captured automatically.

🔗 **[Install from Chrome Web Store](https://chromewebstore.google.com/detail/cainjgmlfcfmggnjagoifenmdlfnhnhe)**  
🌐 **[Website](https://ikarthikmb.github.io/fixquote/)**

---

## What it does

FixQuote lives in your Chrome sidebar. As you browse supplier sites, click to add parts — name and price are captured automatically. Labour, tax, shipping, and markup are calculated live. When you're done, print a clean estimate or copy it to paste into a message or invoice.

## Supported sites (one-click capture)

- Amazon
- eBay
- PhoneLCDParts
- MobileSentrix

Use **Grab from Page** on any other site (Back Market, Facebook Marketplace, iFixit, etc.)

## Features

- One-click part capture from supported sites
- Smart device grouping (iPhone, Samsung, iPad, MacBook, and more)
- Per-part labour, shipping, tax %, and markup
- Customer job names and job history
- Print or copy estimate
- Cloud sync (sign in to access across devices)
- Guest mode — no account required

## Who it's for

Phone & tablet repair shops, laptop technicians, console repair, IT support, appliance techs, and independent freelancers who need to quote parts and labour fast.

## Privacy

Your data stays on your device in guest mode. If signed in, data syncs to your private Firestore document — never shared.  
[Privacy Policy](https://ikarthikmb.github.io/fixquote/#privacy)

## Tech stack

- Chrome Extension Manifest V3
- Firebase Authentication + Firestore (REST API, no SDK)
- Vanilla JS, no build step required
